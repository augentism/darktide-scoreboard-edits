## How to Install This Mod

### Step 1: Download the mod files

1. On the GitHub page, click the green **"Code"** button in the top right.
2. Click **"Download ZIP"**.
3. Once downloaded, find the ZIP file (usually in your **Downloads** folder) and open it.
4. Open the folder inside — you should see a `scripts` folder and a `scoreboard.mod` file.

### Step 2: Find your mods folder

1. Open your game's **mods** folder on your computer.
2. Find the folder for this specific mod and open it.

### Step 3: Replace the files

1. Copy the `scripts` folder and `scoreboard.mod` file from the downloaded ZIP.
2. Paste them into the mod's folder, replacing the existing `scripts` folder and `scoreboard.mod` file when prompted.

That's it — the mod is now updated/installed!

> **Tip:** If you're not sure where your mods folder is located, right click darktide in steam > browse local files.
